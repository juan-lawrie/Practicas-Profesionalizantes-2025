#Comandos Django
-Hacer migraciones: python manage.py makemigrations
- Ejecutar migraciones: python manage.py migrate (o tambien puede ser python3 manage.py migrate)
- Crear superusuario: python manage.py createsuperuser(o tambien puede ser python3 manage.py createsuperuser)
- Ejecutar servidor: python manage.py runserver (o tambien puede ser  python3 manage.py runserver)

Asignar role al superusuario en django:

from api.models import User, Role
superuser = User.objects.get(username='NOMBRE_DEL_SUPERUSUARIO')
gerente_role = Role.objects.get(name='Gerente')  # Ajusta el campo si es diferente
superuser.role = gerente_role
superuser.save()

Verificar que se asigno correctamente el rol:

from api.models import User
superuser = User.objects.get(is_superuser=True)
print(superuser.username, superuser.role)
